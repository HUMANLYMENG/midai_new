# Midai

Midai is a music collection and discovery website where users can import their album collections, view related information (release date, genre, artist, etc.), and display the relationships between albums through an interactive graph (such as common genres between different albums). The website can also recommend new albums based on selected genres. The design style is minimalist and smooth, including album lists, album information, album covers, and interactive graphs.

## Features

1. **Add Albums**:
   - Albums can be added by manually entering album information.
   - CSV files can be uploaded to batch import album information.

2. **View Album Collection**:
   - The "Collection" page displays all the added album information.

3. **Interactive Graph**:
   - The "Collection" page features an interactive graph that shows the relationships between different albums and genres.

## Contents

- `app`: Main directory of the application, containing the initialization, model definitions, route definitions, as well as static resources and template files.
- `instance`: Instance folder, storing instance-level configuration and database files, suitable for configurations and data that need to be separated from the codebase.
- `migrations`: Database migration folder, containing Alembic configuration and migration scripts for managing database versions and schema changes.
- `uploads`: Uploads directory, storing user-uploaded CSV files.

## Script Descriptions

- `csv_to_db.py`: Used to import album information from CSV files into the database.
- `brainz_genres.py`: Updates album genre information using MusicBrainz.
- `discog_genres.py`: Updates album genre information using Discog.
- `brainz_date.py`: Retrieves album release dates from external MusicBrainz.

## File Descriptions

- `config.py`: Configuration file, containing database URI and other configuration information.
- `requirements.txt`: Project dependency list. Generated by `pip freeze > requirements.txt`
- `run.py`: Script to start the Flask application.
